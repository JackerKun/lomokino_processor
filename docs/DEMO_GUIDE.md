## 功能演示步骤

### 1️⃣ 添加图片

**方式一: 拖拽添加**
```
📂 文件浏览器          ➡️     🖥️ GUI 应用
┌──────────────┐              ┌──────────────┐
│ 📷 photo1.jpg│───拖拽────▶  │   图片列表    │
│ 📷 photo2.jpg│              │  [已添加 ✓]   │
│ 📷 photo3.jpg│              └──────────────┘
└──────────────┘
```

**方式二: 按钮添加**
```
点击 [➕ 添加图片] ➡️ 文件选择对话框 ➡️ 选择文件 ➡️ 添加到列表
```

└──────────┘          └──────────┘

## 实际使用示例

### 示例 1: 处理胶片扫描

```
1. 添加扫描的胶片图片
   📷 film_001.jpg
   📷 film_002.jpg
   📷 film_003.jpg

2. 调整裁切去除胶片孔
   上边距: 5%
   下边距: 5%
   左边距: 18%  ← 去除左侧胶片孔
   右边距: 18%  ← 去除右侧胶片孔

3. 设置帧率
   FPS: 12  ← 胶片风格

4. 生成视频
   输出: lomokino_film.mp4
```

### 示例 2: 制作延时视频

```
1. 添加延时照片
   📷 timelapse_0001.jpg
   📷 timelapse_0002.jpg
   ...
   📷 timelapse_0100.jpg

2. 按时间顺序排列 (拖拽调整)
   ✓ 已按文件名排序

3. 裁切构图
   上边距: 10%  ← 去除天空
   下边距: 15%  ← 去除前景
   左边距: 0%
   右边距: 0%

4. 设置帧率
   FPS: 30  ← 流畅播放

5. 生成视频
   输出: timelapse.mp4
```

### 示例 3: 照片幻灯片

```
1. 添加照片
   📷 vacation_01.jpg
   📷 vacation_02.jpg
   📷 vacation_03.jpg

2. 调整展示顺序
   (拖拽到喜欢的顺序)

3. 不裁切
   所有边距: 0%

4. 设置低帧率 (每张显示更久)
   FPS: 1  ← 每张显示 1 秒

5. 生成视频
   输出: slideshow.mp4
```

## 快捷操作

| 操作 | 方法 | 快捷方式 |
|------|------|----------|
| 添加图片 | 点击按钮 | 或拖拽文件到窗口 |
| 删除图片 | 点击"移除选中" | 先选中再点击 |
| 预览图片 | 点击列表中的图片 | 单击即可 |
| 调整顺序 | 拖拽图片 | 在列表中拖动 |
| 重置参数 | 滑块拖到 0 | 恢复默认值 |

## 界面元素说明

### 颜色编码
- 🔵 蓝色按钮 = 常规操作 (添加、移除)
- 🔴 红色按钮 = 危险操作 (清空列表)
- 🟢 绿色按钮 = 主要操作 (生成视频)
- 🟢 绿色边框 = 裁切预览区域

### 状态提示
- ⏳ 进度条显示 = 正在处理
- ✅ 成功提示 = 操作完成
- ❌ 错误提示 = 需要修正

## 性能提示

| 图片数量 | 处理时间 | 建议 |
|---------|---------|------|
| 1-10 张 | < 10 秒 | 快速处理 |
| 10-50 张 | 10-30 秒 | 正常等待 |
| 50-100 张 | 30-60 秒 | 可以喝杯咖啡 ☕ |
| 100+ 张 | > 1 分钟 | 建议分批处理 |
