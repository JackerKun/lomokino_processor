# GUI 版本快速启动

## 🚀 3 步开始使用 GUI 版本

### Mac 用户

```bash
# 1. 安装依赖
./install_gui.sh

# 2. 运行应用
./run_gui.sh

# 3. (可选) 打包独立应用
./build_app.sh
```

### Windows 用户

```batch
REM 1. 安装依赖
install_gui.bat

REM 2. 运行应用
run_gui.bat

REM 3. (可选) 打包独立应用
build_app.bat
```

## 📖 详细文档

请查看 [GUI_README.md](GUI_README.md) 获取完整使用说明。

## ✨ 主要功能

- 拖拽添加和排序图片
- 实时预览裁切效果
- 自定义视频参数 (帧率、裁切边距)
- 进度显示
- 跨平台支持 (Mac/Windows)

## 🎬 使用流程

1. 添加图片 (拖拽或点击按钮)
2. 调整图片顺序 (拖拽排序)
3. 设置裁切参数 (滑块调节)
4. 点击"生成视频"
5. 选择保存位置
6. 等待完成
