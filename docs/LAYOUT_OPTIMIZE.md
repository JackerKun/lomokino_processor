# 🎨 帧显示布局优化

## 问题
提取的帧布局太松散，图片和标签（帧1,2,3）距离太远

## 解决方案

### 1. **调整容器间距**
```python
container_layout.setContentsMargins(2, 2, 2, 2)  # 从(0,0,0,0)改为(2,2,2,2)
container_layout.setSpacing(2)                    # 图片和标签之间只有2px间距
```

### 2. **固定标签高度**
```python
label.setFixedHeight(16)  # 固定为16px，避免自动拉伸
```

### 3. **固定容器尺寸**
```python
container.setFixedSize(126, 140)  # 每个帧卡片固定尺寸
# 120 (图片) + 2 (上边距) + 2 (间距) + 16 (标签) + 2 (下边距) = 142
```

### 4. **优化网格布局**
```python
self.frames_layout.setSpacing(6)              # 网格间距6px
self.frames_layout.setContentsMargins(5,5,5,5) # 容器边距5px
self.frames_layout.setAlignment(AlignTop | AlignLeft)  # 左上对齐
```

## 效果对比

### 优化前
```
┌──────────┐
│          │
│  [图片]   │  ← 120px
│          │
│          │  ← 大量空白
│          │
│   帧 1    │  ← 标签
│          │
└──────────┘
总高度: ~180px
```

### 优化后
```
┌─────────┐
│ [图片]   │  ← 120px
│  帧 1    │  ← 标签(只有2px间距)
└─────────┘
总高度: 140px
```

## 数值说明

| 项目 | 数值 | 说明 |
|------|------|------|
| 图片尺寸 | 120x120 | 缩略图显示区域 |
| 标签高度 | 16px | 固定高度 |
| 上/下边距 | 2px | 容器内边距 |
| 图片-标签间距 | 2px | 非常紧凑 |
| 网格间距 | 6px | 卡片之间的间距 |
| 容器总高度 | 140px | 2+120+2+16+2 (已优化) |

## 整体布局改进

### 紧凑度提升
- **空间利用率**: 提升约 22% ((180-140)/180)
- **视觉密度**: 更紧凑，一屏显示更多帧
- **可读性**: 图片和标签紧密关联，更清晰

### 对齐优化
- 左上对齐，从左到右、从上到下排列
- 固定尺寸确保网格整齐
- 避免不必要的拉伸和空白
