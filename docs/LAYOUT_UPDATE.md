# 🎯 GUI 布局更新说明

## ✨ 新布局特点

### 左右布局设计
```
┌──────────┬───────────────┬────────────────┬──────────┐
│ 胶片列表  │  胶片预览(竖)  │  提取的帧(网格) │ 控制面板 │
└──────────┴───────────────┴────────────────┴──────────┘
```

### 优势
- ✅ **胶片预览竖向显示** - 更适合竖条胶片的查看
- ✅ **左右布局** - 胶片预览和提取的帧并排显示，一目了然
- ✅ **更大的预览区域** - 竖向预览可以显示更长的胶片条
- ✅ **更好的空间利用** - 充分利用宽屏显示器

## 📐 尺寸调整

### 胶片预览区域
- 最小宽度: 200px
- 最大宽度: 400px
- 最小高度: 400px
- 自动保持宽高比

### 提取的帧区域
- 占据更多空间 (stretch=2)
- 网格显示，每行 4 个缩略图
- 可滚动查看所有帧

### 窗口尺寸
- 最小尺寸: 1500 x 900 (增大以适应左右布局)
- 推荐全屏使用以获得最佳体验

## 🔄 使用体验改进

### 查看流程
1. **选择胶片** → 左侧竖向显示完整胶片条
2. **提取帧** → 右侧网格显示所有提取的帧
3. **对比查看** → 可以同时看到原胶片和提取结果

### 操作便利性
- 无需上下滚动即可看到完整胶片预览
- 提取的帧在旁边一目了然
- 更直观的对比效果

## 💡 使用建议

1. **最佳显示器配置**
   - 推荐 1920x1080 或更高分辨率
   - 16:9 或更宽的屏幕比例

2. **查看竖长胶片**
   - 胶片预览会自动适应竖向比例
   - 保持宽高比显示，不会变形

3. **多帧检查**
   - 提取后可以逐个查看右侧的帧缩略图
   - 对照左侧原图确认提取效果
