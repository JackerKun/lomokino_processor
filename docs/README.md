## 按主题浏览

### 🎬 新手入门

2. 选择 GUI 版本或命令行版本
3. 按步骤安装和使用
4. 遇到问题查看 [FAQ](FAQ.md)

### 🔧 解决问题

**帧检测不准确**:
- 查看 [智能检测系统](SMART_DETECTION.md)
- 学习如何调整参数
- 尝试手动选择框模式

**帧画面不完整**:
- 查看 [裁切修复说明](CROPPING_FIX.md)
- 了解 v2.0 的改进
- 使用最新版本

**提取位置不准**:
- 查看 [坐标系统修复](COORDINATE_FIX.md)
- 确保使用 v3.1+

### 📈 高级功能

**精确控制提取**:
- 阅读 [手动选择框](MANUAL_SELECTION_V2.md)
- 学习 8 方向调整
- 使用复制和追加功能

**查看和管理帧**:
- 阅读 [帧查看器](FRAME_VIEWER.md)
- 使用大图查看功能
- 学习键盘快捷键
